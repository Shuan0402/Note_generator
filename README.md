# Note_generator
簡易的手寫筆記生成器

## Prepare in advance
安裝 python 3.6
模型 tkinter、cv2

## Instructions for use
1. 在 Source 開啟小黑窗。
2. 輸入 python -X -u utf8 Note_Generator.py (若無編碼問題可直接運行 Note_Generator.py)。
3. 選取要進行轉換的文件名稱，點選 select。
4. 確認完文件內容後，點選 Yes。
5. 預覽所需的字符，點選 Upload 開始上傳字符圖片檔。
6. 一字可上傳多字跡，點選 next 可進到下一個字符進行上傳。
